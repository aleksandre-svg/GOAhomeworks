CSS Positioning Explained

Learn how different position values work in CSS.

Position: Static

br

Position: static (default)

By default, all elements are positioned statically. This means they are positioned according to the normal flow of the document.

Position: Relative

Position: relative (offset relative to its normal position)

Position: relative with offset

Position relative allows you to move an element relative to its normal position using the top, right, bottom, and left properties.

Position: Absolute

Position: absolute (positioned relative to the nearest positioned ancestor)

Position: absolute inside relative

Absolute positioning removes an element from the normal flow and positions it relative to its nearest positioned ancestor.

Position: Fixed

Position: fixed (fixed on the screen, independent of scrolling)

Fixed positioning allows an element to be fixed in place on the screen even when the page is scrolled.

Position: Sticky

Position: sticky (scrolls with the page until a certain point)

Sticky positioning allows an element to scroll with the page until it reaches a defined position, where it then becomes "stuck" in place.
