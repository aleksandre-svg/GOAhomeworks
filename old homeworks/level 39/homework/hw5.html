What is HTML?

HTML stands for HyperText Markup Language. It's the standard language for creating web pages. HTML describes the structure of a webpage using a system of tags and attributes. Every webpage you visit is built using HTML.

Learning HTML is the first step to becoming a web developer. It allows you to create the foundation of any website, from basic structures to complex layouts.

Basic HTML Tags

HTML tags are the building blocks of a webpage. Here are a few essential tags you'll start with:

  • <html> - The root element of an HTML document
  • <head> - Contains metadata and links to external resources
  • <body> - Contains the content of the webpage
  • <h1>, <h2>, <h3> - Headings used to define sections
  • <p> - Paragraph tag for text content
  • <a> - Anchor tag for links

Start Coding HTML

To get started, open a simple text editor (like Notepad or Sublime Text) and create a new file. Save it as index.html.

Here's a basic example of an HTML page structure:

<!DOCTYPE html> <html> <head> <title>My First Page</title> </head> <body> <h1>Welcome to Learning HTML!</h1> <p>This is a paragraph of text.</p> </body> </html>
